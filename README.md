# The Blog

A static blog site built with Hugo and Tailwind CSS, deployed on Netlify.

## Features

- Fetches 100 blog posts from the [JSONPlaceholder API](https://jsonplaceholder.typicode.com/posts)
- Responsive grid layout (1 / 2 / 3 columns)
- Live search bar that filters posts by title
- Modal that shows full post details on click or "Read More"
- Contact form processed by Netlify Forms with email notifications via Zapier
- Well-structured Hugo partials for reusable components

## Tech Stack

- [Hugo](https://gohugo.io/) — static site generator
- [Tailwind CSS](https://tailwindcss.com/) — utility-first CSS (via CDN)
- [JSONPlaceholder](https://jsonplaceholder.typicode.com/) — mock REST API
- [Netlify](https://netlify.com/) — hosting and form handling
- [Zapier](https://zapier.com/) — form submission email notifications
